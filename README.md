Discord theme that makes UI elements 'pop out' on hover.

# Preview

[Video](https://i.imgur.com/1fvOejU.mp4)

# Installation

## Powercord

1. Run this in CMD: `cd powercord/src/Powercord/themes && git clone https://github.com/devinkid1/Pop/`

## BetterDiscord

1. Download the theme through this link: https://betterdiscord.net/ghdl?id=3489
2. Move the downloaded file to your themes folder.

## Stylus

(You will need to know how to make a theme for Stylus)

1. Go to https://discord.com, open Stylus, and then press "Write style for: discord.com".
2. Name the style "Pop!" or something else if you want.
3. Copy and paste the CSS code from [here](https://raw.githubusercontent.com/devinkid1/Pop/main/Pop.theme.css) into the right box.
4. Save the style.
